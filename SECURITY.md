# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in FlexProof, please report it
responsibly by emailing **security@codaex.com** (or open a private advisory
on GitHub).

Please do NOT open a public issue for security vulnerabilities.

## Scope

This policy covers the FlexProof codebase including:
- Cryptographic commitment and signature modules
- Artifact issuance and verification logic
- API endpoints and data flow boundaries

## Response

We will acknowledge receipt within 48 hours and aim to provide a fix or
mitigation within 14 days for confirmed vulnerabilities.
